# Goal:
- Create for reinforcement learning the app with the name `frozen_lake` with the gym environment `FrozenLake-v1`, to run live with selectable methods and compare methods with a live reward plot.

## Very important: 
- !!! use `..\rl_init.md` for inital requirements !!!

## Features
- Selectable map-names: 4x4, 8x8 (default 4x4)
- Selectable slippery mode: IsSlippery (default false)
- Selectable methods: q-learning, sarsa, expected sarsa
