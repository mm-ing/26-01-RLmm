# Gridworld RL demo

Tkinter-based Gridworld reinforcement learning demo with multiple algorithms and plotting.

## Files
- grid_logic.py — GridWorld environment, policies, Agent, and pathfinding helpers.
- grid_gui.py — GUI for configuration, training, plotting and table dialogs.
- grid_app.py — Entry point.
- grid_sample.md — Prompt.

## Requirements
- Python 3.9+
- matplotlib

## Run

```bash
python grid_app.py
```
