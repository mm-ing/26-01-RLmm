# Goal:
- Create for reinforcement learning the app with the name `lunar_lander` with the gym environment `LunarLander-v3`, to run live with selectable methods and compare methods with a live reward plot.

## Very important: 
- !!! use `..\rl_init.md` for inital requirements !!!

## Additional Features
- Selectable methods (dropdown): DQN, DDQN, Prioritized DDQN, Dueling DDQN, Distributional DQN, Noisy DQN, Rainbow
- Selectable environment parameters: continous (default: false), gravity (default: -10.0), enable_wind (default: false), wind-power (default: 15.0), turbulence-power (default: 1.5)
