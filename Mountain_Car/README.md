# Mountain Car RL demo

Tkinter-based MountainCar-v0 reinforcement learning demo with multiple DQN variants, Rainbow, and live plotting.

## Files
- mountain_car_logic.py — environment wrapper, policies, and agent.
- mountain_car_gui.py — GUI for configuration, training, animation, and plots.
- mountain_car_app.py — entry point.
- mountain_car.md — prompt.

## Requirements
- Python 3.9+
- gymnasium
- numpy
- matplotlib
- pillow
- torch

## Run

```bash
python mountain_car_app.py
```
