# Pendulum RL demo

Tkinter-based Pendulum-v1 reinforcement learning demo with multiple DQN variants and live plotting.

## Files
- pendulum_logic.py — environment wrapper, policies, and agent.
- pendulum_gui.py — GUI for configuration, training, animation, and plots.
- pendulum_app.py — entry point.
- pendulum.md — prompt.

## Requirements
- Python 3.9+
- gymnasium
- numpy
- matplotlib
- pillow
- torch

## Run

```bash
python pendulum_app.py
```
