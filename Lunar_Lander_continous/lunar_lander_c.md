# Goal:
- Create for reinforcement learning the app with the name `lunar_lander_c` with the gym environment `LunarLander-v3` and option contious = true, to run live with selectable methods and compare methods with a live reward plot.

## Very important: 
- !!! use `..\rl_init.md` for inital requirements !!!

## Additional Features
- Selectable methods (dropdown): Rainbow, A2C, TRPO, PPO
- Selectable environment parameters: continous (default: true), gravity (default: -10.0), enable_wind (default: false), wind-power (default: 15.0), turbulence-power (default: 1.5)
